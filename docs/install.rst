Install
=======

To install ``multinorm`` use pip::

    pip install multinorm

This is a single-file (``multinorm.py``) module, so if you want,
you can also just download it from
https://github.com/cdeil/multinorm/blob/master/multinorm.py

::

    curl -O https://github.com/cdeil/multinorm/raw/master/multinorm.py

Dependencies
------------

This package supports Python 3.6 or later.

Required dependencies: `numpy`_ and `scipy`_.

Optional dependencies (used by a few methods):

- `pandas`_
- `matplotlib`_
- `xarray`_
